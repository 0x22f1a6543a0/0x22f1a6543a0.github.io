Smarterjson介绍

一个python库，基于python标准发行版中的json库；

提供了部分算法以帮助更多json需求！

Python安装smarterjson

pip install smarterjson

pip下载慢
临时使用[清华镜像站]

pip install smarterjson -i https://pypi.tuna.tsinghua.edu.cn/simple some-package

永久使用[清华镜像站]

再次之前您需要更新pip(>=10.0):

python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --upgrade pip

下载库

pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

smarterjson使用

库基础

write函数

函数变量

__value__为需要写入的值

fp为文件路径

encoding为编码

ensure_ascii表示编码时是否将非ASCII字符转换为unicode码

indent为写入缩进

用途

写入指定的json文件，相同值替换

append函数

函数变量

__value__为需要追加值

fp为文件路径

encoding为编码

ensure_ascii表示编码时是否将非ASCII字符转换为unicode码

indent为写入缩进

用途

追加数据在指定的json文件下；相同值则替换

read函数

函数变量

__content__为需要读取键的索引

fp为文件路径

encoding为编码

return_type为返回类型

用途

根据__content__来读取json文件

返回类型为str时则返回字符串

返回类型为list时则返回列表

返回类型为int时则返回长度

返回类型为dict时则返回字典

exist函数

函数变量

__key__为需要查找的键

fp为文件路径

encoding为编码

用途

根据fp来查找是否由指定的键存在于json文件中，返回布尔值

revise函数

函数变量

__content__为需要索引

__value__为需要写入的值

fp为文件路径

encoding为编码

ensure_ascii表示编码时是否将非ASCII字符转换为unicode码

indent为写入缩进

用途

根据__content__和fp来修改json中的值，值为__value__
